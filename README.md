# Lumina — Smart Desk Lamp Landing Page

A modern, responsive landing page for **Lumina**, the world's first AI-powered desk lamp that adapts to your mood, schedule, and rhythm automatically.

## 🌟 Features

- **AI Mood Detection** — Lumina learns your patterns and adjusts brightness and color temperature throughout the day
- **Circadian Rhythm Mode** — Warm light in the evening, cool blue-white during focus hours
- **App Control** — Full control from your phone with HomeKit and Google Home integration
- **Energy Efficient** — LED technology uses 80% less energy, lasts 25,000 hours
- **Voice Control** — Compatible with Alexa, Google Assistant, and Siri
- **1000+ Scenes** — Pre-curated lighting scenes or create your own

## 📋 Technical Specifications

| Specification | Details |
|---|---|
| Brightness | 50–1000 lux |
| Color Temperature | 2200K–6500K |
| Connectivity | Wi-Fi 6 + Bluetooth 5.2 |
| Power Consumption | 12W max |
| LED Lifespan | 25,000 hours |
| App Compatibility | iOS 16+, Android 12+ |

## 🚀 Getting Started

### View Locally

1. Clone or download this repository
2. Open `landing_page.html` in your web browser
3. Navigate through the page using the navigation menu

### File Details

- **landing_page.html** — Contains the complete page structure with:
  - Navigation bar
  - Hero section with CTA buttons
  - Features grid
  - Technical specifications
  - Pricing section

- **style.css** — Modern, responsive styling featuring:
  - Dark theme with gold accents
  - Smooth scrolling behavior
  - Responsive grid layouts
  - Animations and hover effects
  - CSS custom properties for easy theming

## 🛠️ Technologies Used

- **HTML5** — Semantic markup
- **CSS3** — Modern styling with custom properties and animations
- **Google Fonts** — Syne and Inter typefaces
- **Responsive Design** — Mobile-first approach


## 📝 Customization

### Change Product Information

Edit the relevant sections in `landing_page.html`:
- Update pricing in the pricing section
- Modify features and descriptions
- Change product images and links

### Modify Theme Colors

Edit CSS custom properties in `style.css`:
```css
:root {
  --bg: #07080a;           /* Background */
  --gold: #e8c84a;         /* Primary accent */
  --text: #f0ede8;         /* Text color */
  /* ... more variables */
}
```
